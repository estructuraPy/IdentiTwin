Overview
--------

The ``report_generator`` module creates text-based reports summarizing system configuration, monitoring session details, and detected events.

Key Features
-----------

* System configuration report: Documents settings, sensor setup, rates, etc.
* Monitoring summary report: Provides session statistics, performance metrics, and event summaries.
* Event summarization: Reads information from individual event report files.
* File management: Saves reports to the designated directory.
* Timestamping: Includes generation timestamps.
