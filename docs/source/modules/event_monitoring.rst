Overview
--------

The ``event_monitoring`` module provides real-time event detection and monitoring capabilities:

* Detection of structural events and anomalies
* Threshold-based monitoring
* Event classification and categorization
* Real-time alert generation
* Event logging and reporting

Key Features
-----------

* Configurable event thresholds
* Multiple sensor event correlation
* Automatic event logging
* Alert notification system
* Historical event analysis
* Event severity classification
